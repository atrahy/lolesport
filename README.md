## Installation
> [!IMPORTANT]
> For the best experience a [Nerd Font](https://www.nerdfonts.com/) installed and enabled is required.

### Package manager

```bash
# macOS
brew install rift
```

### Go

```bash
go install github.com/matthieugusmini/rift@latest
```

### Build

```bash
git clone https://github.com/matthieugusmini/rift.git
cd rift
go build
```
